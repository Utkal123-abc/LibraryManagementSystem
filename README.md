#  Library Management System

##  Project Overview
This **Library Management System** is a basic Java application that helps in managing books in a library ‚Äî from adding and removing books to issuing and returning them. The project is developed as part of a self-learning journey into **Java and database connectivity**.

##  Features
-  Add, delete, and update book details
-  Search for books
-  Issue and return books
-  MySQL integration for persistent data storage
